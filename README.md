## Overview
This project is a cyber security practical lab using **Kali Linux** and
**Metasploitable** in a VirtualBox environment.

The lab demonstrates how vulnerabilities can be identified and exploited
using the **Metasploit Framework** for learning purposes.

---

## Learning outcomes
- Setting up Kali Linux and Metasploitable in VirtualBox  
- Checking IP addresses and network connectivity  
- Scanning open ports on a target system  
- Using Metasploit to exploit a vulnerability  
- Getting remote shell access  
- Encrypting files using GPG  
- Hiding data inside an image using steganography  
- Transferring files to a remote machine  

---

## Tools Used
- Kali Linux  
- Metasploitable  
- Metasploit Framework  
- Nmap  
- GPG  
- Steghide  
- Netcat
